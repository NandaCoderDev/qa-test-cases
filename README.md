# 🧪 QA - Casos de Teste e Relatório de Bugs

Projeto com foco em Qualidade de Software (QA), contendo documentação de testes e identificação de falhas em sistema de login.

---

## 🧠 Sobre o projeto
Este projeto foi desenvolvido com o objetivo de praticar a criação de casos de teste e documentação de bugs, simulando atividades reais da área de QA.

---

## 📊 Conteúdo

### ✔️ Casos de Teste
Arquivo contendo cenários de teste para funcionalidade de login:
- Login com dados válidos
- Login com senha incorreta
- Validação de campos obrigatórios

### 🐞 Bug Report
Arquivo contendo registro de falhas identificadas:
- Login permitindo senha vazia
- Possíveis inconsistências de validação
