# 🧪 QA Test Cases Project

Practical software testing project focused on validating functionalities, creating test cases, and identifying potential issues.

## 🎯 Objective
Simulate the work of a QA Analyst by testing a login system, covering both positive and negative scenarios.

## 🔍 Types of Testing Performed
- Functional testing  
- Exploratory testing  
- Field validation  
- Positive and negative scenarios  

## 📋 Test Cases
Includes test scenarios such as:
- Login with valid credentials  
- Login with invalid password  
- Empty required fields  
- Password validation rules  

## 🐞 Bug Report
Documentation of potential issues found during testing, including steps to reproduce, expected results, and actual results.

## 🔐 Security
Application of basic security concepts based on the OWASP Top 10.
